# Generate pseudo-label from CT scanned images
This repository hosts the source code for the model-free pseudo-label generation from CT scanned images, as discussed in the paper **"Model-Free Pseudo-Label Generation for Material CT Image Segmentation."**

This work is currently under review at the Computational Materials Science journal. DOI will be added here after the review process.
